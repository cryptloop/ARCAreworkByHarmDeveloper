## ⚙️ Перед деплоем — настроить формы (1 минута)

Сейчас формы не отправляют заявки, пока не вставлен ключ доступа.

1. Откройте **[web3forms.com](https://web3forms.com)** — бесплатно, без регистрации.
2. Введите свою почту → получите ключ доступа (Access Key).
3. В `index.html` найдите **два** поля:
   ```html
   <input type="hidden" name="access_key" value="YOUR_ACCESS_KEY_HERE">
   ```
   (одно в hero-форме, второе в секции контактов)
4. Замените `YOUR_ACCESS_KEY_HERE` на свой ключ — в обоих местах.
5. Заявки начнут приходить на вашу почту.

> Письма приходят с темы, указанной в `<input name="subject">`. Бесплатный лимит — 250 заявок/месяц.

## 🚀 Деплой на Vercel

### Способ 1 — через Git (рекомендуется, авто-обновление при пуше)

```bash
# 1. Инициализируйте репозиторий
cd ARCA-landing
git init
git add .
git commit -m "ARCA landing"

# 2. Залейте на GitHub (создайте пустой репозиторий github.com/new)
git remote add origin https://github.com/ВАШ_ЛОГИН/arca-landing.git
git branch -M main
git push -u origin main
```

3. Зайдите на **[vercel.com](https://vercel.com)** → Login with GitHub
4. **Add New → Project** → выберите репозиторий `arca-landing`
5. Настройки (Vercel определит сам, но проверьте):
   - **Framework Preset:** `Other`
   - **Build Command:** *(оставить пустым)*
   - **Output Directory:** *(оставить пустым)*
6. **Deploy** → через ~20 секунд готово: `arca-landing.vercel.app`

### Способ 2 — через CLI (быстрее, без Git)

```bash
npm i -g vercel
cd ARCA-landing
vercel          # превью-ссылка
vercel --prod   # боевая ссылка
```

## 🔧 Что можно донастроить

| Что | Где |
|---|---|
| Телефон / адрес / почта | `index.html` (поиск по `+7 343`) |
| Цены тарифов | секция `#tariffs` |
| Цены калькулятора | `data-price` в кнопках тарифов, `data-add` в опциях |
| Ссылки на соцсети | футер `footer__social` |
| Реальные фото объектов | папка `images/` + пути в `index.html` / `styles.css` |
